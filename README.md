# This client was discontinued for several reasons. All of original contributors has moved on to other projects & things ([**Fogma**](https://github.com/fogmalol), [**Phyrobyte**](https://github.com/phyrobyte), [**Somesable**](https://github.com/somesable)), and we are no longer able to maintain support for the latest versions, also the client is not yet ready for release due to horrible auth, missing auto updater etc

# That said, the project is now community-driven.







<h1 align="center">Toba Client</h1>

<p align="center">
  A Minecraft 1.21.11 Fabric utility client mod.
</p>


## Requirements

- Minecraft **1.21.11**
- [Fabric Loader](https://fabricmc.net/) **0.18.4+**
- [Fabric API](https://modrinth.com/mod/fabric-api) **0.141.2+**
- Java **21**


```bash
git clone https://github.com/phyrobyte/Toba-Client.git
cd Toba-Client
./gradlew build
```

The built jar will be in `build/libs/`.
